# @qlippy/opencode

OpenCode plugin for the Qlippy desktop pet.

## Install

```bash
npm install -g @huangcheng/qlippy-gateway
cd ~/.config/opencode
npm install @huangcheng/qlippy-opencode
```

Then add to your `opencode.json`:

```json
{
  "plugin": [
    "@qlippy/opencode"
  ]
}
```

Restart OpenCode and it will auto-load the plugin.
