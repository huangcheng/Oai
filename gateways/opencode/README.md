# @clippy/opencode

OpenCode plugin for the Clippy desktop pet.

## Install

```bash
npm install -g @eastlate/clippy-gateway
cd ~/.config/opencode
npm install @eastlate/clippy-opencode
```

Then add to your `opencode.json`:

```json
{
  "plugin": [
    "@clippy/opencode"
  ]
}
```

Restart OpenCode and it will auto-load the plugin.
