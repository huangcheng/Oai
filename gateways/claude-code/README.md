# @clippy/claude-code

Claude Code hooks for the Clippy desktop pet.

## Install

```bash
npm install -g @eastlate/clippy-gateway
npm install -g @clippy/claude-code
npx @eastlate/clippy-claude-code
```

The last command merges the hooks into `~/.claude/settings.json`.

## Manual install

If you prefer to merge manually, copy `settings.json` from this package into your `~/.claude/settings.json` under the `"command"` key.
