# @qlippy/claude-code

Claude Code hooks for the Qlippy desktop pet.

## Install

```bash
npm install -g @huangcheng/qlippy-gateway
npm install -g @qlippy/claude-code
npx @huangcheng/qlippy-claude-code
```

The last command merges the hooks into `~/.claude/settings.json`.

## Manual install

If you prefer to merge manually, copy `settings.json` from this package into your `~/.claude/settings.json` under the `"command"` key.
